05 · Pipeline

Flow at a glance

The five-stage pipeline that maps directly to the deliverable: ingest brief → resolve assets → generate missing → compose per ratio → verify.

01
Ingest brief
Parse JSON brief: brand, products, markets, audience, message.
02
Resolve assets
Reuse local assets where present; flag gaps.
03
Generate
Fill missing hero images via GenAI image API.
04
Compose
Render 1:1, 9:16, 16:9 with localized message.
05
Verify
Brand & legal compliance checks; report.
06 · Brief schema

Campaign brief

The input contract. Anything renderable downstream comes from this. Two products minimum; one brief produces N×3 creatives where N = product count.

brief.json Valid
Campaign
Summer Refresh 2026
Brand
sparkling-co
Products
Sparkling Citrus · Sparkling Berry
Markets
us-en · mx-es
Audience
18–34, urban, health-conscious
Message
"Crack open something brighter."
Ratios
1:1 · 9:16 · 16:9

Example JSON

{
  "campaign": "summer-refresh-2026",
  "brand": "sparkling-co",
  "products": [
    { "name": "Sparkling Citrus", "sku": "SPK-CIT-12" },
    { "name": "Sparkling Berry", "sku": "SPK-BRY-12" }
  ],
  "markets": ["us-en", "mx-es"],
  "audience": "18-34, urban, health-conscious",
  "message": {
    "en": "Crack open something brighter.",
    "es": "Abre algo más brillante."
  },
  "ratios": ["1x1", "9x16", "16x9"]
}
